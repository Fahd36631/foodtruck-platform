# Project Charter – Portfolio Project  
## Team Name: Truck  

---

# 1. Project Overview

## Purpose

The purpose of this project is to develop a mobile application that helps users discover food trucks in real time, explore their menus, and place orders easily.

This platform aims to solve the problem of limited visibility for food trucks and the difficulty users face in locating them when they frequently change locations.

---

## Objectives (SMART)

- Enable users to discover nearby food trucks using an interactive map.  
- Provide real-time tracking of food truck locations.  
- Allow users to place orders and receive order status updates.  

---

# 2. Technology Stack

The project will be developed using the following technologies:

- **Frontend:** React Native  
- **Backend:** Node.js with Express  
- **Database:** MySQL

# 4. Project Scope

## In-Scope (Included Features)

- Display food trucks on a live map  
- Filter food trucks by location and category  
- View food truck menus  
- Place orders and receive order number  
- Online payment system  
- Real-time notifications between user and food truck owner  
- Food truck registration with license upload  

---

## Out-of-Scope (Excluded Features)

- AI-based recommendations  
- Advanced analytics dashboard  
- Multi-language support  
- Delivery system (orders are pickup only)  
- Integration with external food platforms  

---

# 5. Risks and Mitigation

| Risk | Mitigation Strategy |
|------|-------------------|
| Difficulty implementing real-time location tracking | Use reliable APIs like Google Maps |
| Limited experience with payment integration | Use third-party services like Stripe |
| Time constraints | Divide tasks and prioritize MVP features |
| Team coordination issues | Schedule regular meetings and use communication tools |

# 6. High-Level Plan

# 7. Expected Outcome

The project will deliver a functional Minimum Viable Product (MVP) that:

- Helps users easily discover food trucks  
- Enables real-time tracking of food truck locations  
- Allows users to place orders and receive updates  
- Supports food truck owners in increasing visibility and managing orders  

The final product will demonstrate a complete full-stack system with real-world functionality.
